# INF1300-Exercicio_Extra

=== exercício prático de consolidação ===

Crie um aplicativo capaz de listar e cadastrar usuários. A ideia do aplicativo é exibir em sua tela inicial uma lista de usuários já cadastrados e, além disso, também nessa tela, disponibilizar um botão para adicionar um novo usuário.

Para adicionar um usuário, o aplicativo deverá utilizar uma navegação (Navigation) para ir até uma segunda tela. Nessa tela, deve ser possível  adicionar o nome e ano de nascimento de um novo usuário, tendo a opção de salvar a adição ou cancelar a digitação, através de dois botões. Após a cinfirmsr/cancelar, o seu aplicativo deverá voltar para tela inicial através da Navigation e exibir a lista de todos os  usuários contendo tb o novo usuário. Além disso, deveremos ter também a opção de remover um usuário da lista, e  para isso, devemos utilizar do gesto de "arrastar para o lado" (podendo utilizar componentes para tal, como visto nas aulas).

O trabalho deverá ser entregue através de um link de um repositório público no github ou bitbucket por e-mail até as 23:59 do dia 07/04. Será avaliada a corretude da aplicação (conter o que foi descrito), a estrutura do código (utilização de classes statefull/stateless, widgets de maneira correta) e a organização do código.
